HTTP Methods Lab with JavaScript Fetch API

GET Method - Retrieve Data

The GET method is used to retrieve data from the server (read-only).

POST Method - Create Data

The POST method sends data to the server to create a new resource.

PUT Method - Update Entire Resource

The PUT method updates an existing resource by replacing it entirely.

PATCH Method - Partial Update

The PATCH method partially updates an existing resource.

DELETE Method - Remove Resource

The DELETE method removes a specified resource.
